# EPD Skill Registry

AGENTROPOLIS-EPD Skills are atomic contracts governing editorial intelligence and publishing actions.

## Registry

| Skill | Tier | Job |
|---|---|---|
| EPD-SCOUT | core | Discover relevant signals and source material |
| EPD-VERIFY | core | Validate claims, dates, quotations, and provenance |
| EPD-DESK | core | Produce structured editorial drafts from verified research |
| EPD-SEO | extended | Generate search metadata, schema, categories, and tags |
| EPD-MEDIA | extended | Prepare and manage publication media assets |
| EPD-PREVIEW | core | Render a complete pre-publication review package |
| EPD-GATE | infrastructure | Enforce explicit authorization before release |
| EPD-PUBLISH | core | Execute approved publishing actions |
| EPD-WIRE | extended | Distribute released content to approved channels |
| EPD-RECEIPT | infrastructure | Record provenance, approval, action, and destination metadata |

Each Skill must define triggers, dependencies, inputs, outputs, chain relationships, failure states, and one concrete usage example before being marked production-ready.
